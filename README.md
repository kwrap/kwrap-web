# kwrap-web

[Kwrap's Web client](https://web.kwrap.app)

## Dev

```bash
# Clone repo
git clone https://github.com/kwrap/kwrap-web.git --recursive
cd kwrap-web

# Install dependencies
pnpm i

# Run
pnpm dev
```

Open [http://localhost:1234/](http://localhost:1234/)

### Build

```bash
pnpm build
```
